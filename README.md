# Lensify – AI Image Classification Platform

Lensify is a web-based AI application that allows users to upload images and instantly receive predictions using a deep learning model. The platform focuses on providing fast, accurate, and easy-to-understand image recognition.

---

**Live Demo:** https://lensifyai.streamlit.app

---

## Problem

Understanding image content manually can be time-consuming and requires domain knowledge. Users often face:
- Difficulty identifying unknown objects in images  
- Lack of quick AI tools for image understanding  
- Limited access to simple and interactive ML applications  

Lensify solves this by providing an easy-to-use platform for instant image classification.

---

## Solution

Lensify uses a pretrained deep learning model to analyze uploaded images and generate predictions in real time. It provides users with clear results along with confidence scores, making AI more accessible and understandable.

---

## Features

- Image upload and instant prediction  
- Top 3 predictions with confidence scores  
- Fast and efficient processing  
- Clean and interactive UI

## How It Works

1. User uploads an image.  
2. The model preprocesses the image.  
3. AI model analyzes the image.  
4. Top predictions are generated.  
5. Results are displayed with confidence scores.
